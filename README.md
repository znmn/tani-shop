# Tani Shop

Tani-Shop adalah aplikasi penjualan sederhana untuk hasil produk-produk pertanian

## Installation

- Linux:

```bash
apt-get install python3 python3-pip -y
pip3 install -r requirements.txt
python3 main.py
```

- Windows:

  - Install python3 and pip
  - Run file "run.bat"

```bash
python main.py
```
